# Map of Us 开源广告文案

## 核心一句话

很多人想要这个项目，所以我把它开源了：一个本地优先的情侣记忆地图模板，可以把城市、照片和故事都放进自己的地图里。

## 朋友圈 / 小红书

### 版本 A：真诚版

之前发过这个情侣记忆地图，没想到很多人都来问能不能开源。

所以我整理了一版：Map of Us 已经开源了。

它可以记录去过的城市、照片、纪念日和小故事，数据默认存在自己的电脑里，也可以打包成 macOS / Windows 桌面应用。

想做情侣地图、旅行足迹、家庭相册、朋友回忆录的，可以拿去改成自己的版本。

GitHub 已开源，主页/评论区自取。

### 版本 B：产品感版

把两个人的回忆，放回地图上。

Map of Us 是一个开源的本地情侣记忆地图模板：

- 中国地图、省份详情、城市回忆
- 照片、文案、地标、纪念日都能自定义
- 本地优先，数据保存在自己的电脑
- 支持 Electron 桌面端打包

给想做私人记忆应用的人一个起点。

GitHub 已开源。

### 版本 C：轻松版

被催开源催了一阵子，我投降。

这个情侣记忆地图模板已经放出来了：Map of Us。

你可以把它改成你们自己的旅行地图、恋爱纪念册、家庭相册，甚至朋友局足迹图。照片和文案都能换，数据本地保存，还能打包成桌面应用。

想折腾的朋友，源码已在 GitHub。
